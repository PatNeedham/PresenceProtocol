# PresenceProtocol

Real-time safety networks for constitutional rights activists.

## About

PresenceProtocol is a civic engagement platform that coordinates live geolocation tracking and safety notifications for people participating in civil disobedience, legal observation, and constitutional rights activities.

### Key Features

- **Live GPS Tracking** - Real-time location sharing with configurable privacy
- **Automatic Check-ins** - Customizable intervals with automatic alerts if missed
- **Emergency Notifications** - One-tap panic button sends instant alerts
- **Safety Network** - Connect with trusted contacts, legal observers, and attorneys
- **Know Your Rights** - Educational resources about constitutional protections
- **Community Leaderboards** - Track documented civic participation

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Lucide Icons

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/PatNeedham/PresenceProtocol.git
cd PresenceProtocol

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build

```bash
npm run build
```
